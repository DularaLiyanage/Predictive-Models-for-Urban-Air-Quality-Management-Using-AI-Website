# AirQ AI — Research Project Website

## File Structure
```
airq-website/
├── index.html          — Home page
├── domain.html         — Literature survey, objectives, methodology
├── milestones.html     — Project timeline & milestones
├── documents.html      — Downloadable documents
├── presentations.html  — Slide deck links
├── about.html          — Team & supervisor
├── contact.html        — Contact form
├── styles.css          — All styles (single shared CSS file)
├── images/             — Add team photos & project images here
├── documents/          — Add PDF files here
└── presentations/      — Add presentation PDFs here
```

## Quick Start
1. Open `index.html` in any browser — no server needed.
2. Search for `[Add` to find all placeholders and fill them in.
3. Replace document/presentation `href` links with your actual file paths.
4. For the contact form, connect it to Formspree, Netlify Forms, or similar.

## Customisation Tips
- Team photos: Add `images/member1.jpg` etc. and update `<div class="avatar">` in about.html
- Google Map: Replace the grey map box in contact.html with a Google Maps `<iframe>` embed
- Colors: Edit CSS variables at the top of styles.css
- Fonts: Loaded from Google Fonts (DM Sans + Playfair Display)
